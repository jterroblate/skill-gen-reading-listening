## 1. 通用规则 | General Rules

### 1.1 每题的三层结构 | Three-Layer Structure Per Question

每一道题必须包含这三个层次：

```
Layer 1 — Locator (定位信号)
    ↓
    "Now, about the cost..." → 问题: "Fee: ___"
    ↓
Layer 2 — Answer Support (答案支撑)
    ↓
    "The registration fee is £150" → 答案: 150
    ↓
Layer 3 — Paraphrase (转述验证)
    ↓
    题干不能直接复制 "registration fee" → 必须转述为 "Fee"
```

- 缺少任意一层 → 题目不合格
- Locator 必须是自然的话语过渡，不能生硬
- Paraphrase 力度：至少替换 1 个关键词或改变句式

### 1.2 干扰多样性 | Distractor Diversity

- **禁止**同一 Section 内使用超过 3 次相同干扰机制
- 一个 Section 至少使用 4 种不同的干扰类型
- 干扰必须具有表面合理性（不能是明显错误的答案）
- Go to [distractor_library.md](distractor_library.md) for the full mechanism catalog

### 1.3 答案唯一性 | Answer Uniqueness

- 填空题：每个空必须有唯一正确答案
- 选择题：正确选项在排除法下可唯一确定
- Map tasks 禁止 (根据 Cambridge 19/20 分析，map 题型已被淘汰)

### 1.4 Stem 转述原则 | Stem Paraphrase Principle

- Stem 不能直接复制 transcript 中超过 3 个连续单词
- 必须至少替换 1 个关键词或重组句式
- Go to [paraphrase_engine.md](paraphrase_engine.md) for transformation patterns

### 1.5 难度校准 | Difficulty Calibration

| Band Range | Vocab Level | Comprehension | Paraphrase Depth |
|-----------|-------------|---------------|------------------|
| 5.5-6.0 | 日常词汇 | 字面理解 | 简单同义词替换 |
| 6.5-7.0 | 日常+部分学术 | 部分推理 | 词组重组+同义短语 |
| 7.5-8.0 | 学术词汇+习语 | 推理+隐含意义 | 深层语义转述 |

## 2. Section 1 详细规则

### 2.1 场景清单 | Scenario List

Section 1 必须是双人日常对话，场景限于以下类型之一：

| 场景 | 对话角色 | 典型信息交换内容 |
|------|---------|-----------------|
| Accommodation booking | 租客 + 房东/中介 | dates, prices, facilities, location, rules |
| Event registration | 报名者 + 工作人员 | name, contact, preferences, fees |
| Course enrollment | 学生 + 行政人员 | course name, schedule, prerequisites, cost |
| Club membership | 申请人 + 会员经理 | membership type, fees, benefits, personal info |
| Travel arrangement | 旅客 + 旅行社/酒店 | destinations, dates, transport, package |
| Job application | 求职者 + HR | position, experience, availability, salary |
| Health service registration | 患者 + 前台 | personal details, symptoms, appointment |
| Library membership | 申请人 + 图书管理员 | ID, address, membership type, rules |

### 2.2 对话结构规则 | Dialogue Structure Rules

每个 S1 对话必须遵循以下结构：

```
Opening Phase
├── Greeting/self-introduction
└── Purpose statement ("I'm calling about... / I'd like to...")

Information Exchange Phase (多轮 question-answer loop)
├── Question 1 (信息1, may contain first rejection)
│   ├── Ask → Response → Clarification → Confirm
│   └── OR: Ask → First Suggestion → Rejection → Correction → Confirm
├── Question 2 (信息2, similar pattern)
├── ...
└── Question N

Closing Phase
├── Summary of key info (optional)
├── Next steps ("We'll send you...")
└── Thank you / Goodbye
```

### 2.3 答案出现机制 | Answer Appearance Mechanism

**核心规则：答案不能直接给出，必须经过至少一次澄清 (clarification) 或确认 (confirmation) 后才确定。**

**模式模板：**

```
模式 A — 确认型 (Information naturally emerges):
Speaker A: What's your name?
Speaker B: It's Sarah Jones. [初始]
Speaker A: Can you spell that for me? [要求澄清]
Speaker B: Yes, J-O-N-E-S. [确认]
→ 答案确定：Jones

模式 B — 纠正型 (First suggestion rejected):
Speaker A: Would you prefer the Gold membership?
Speaker B: That's too expensive, actually. What about the Basic one?
Speaker A: The Basic plan is £45 per month.
Speaker B: Yes, that works for me.
→ 答案确定：Basic (Gold被拒绝)
```

**注意：至少 50% 的题目需要包含某种形式的澄清或纠正。**

### 2.4 干扰类型 | Distractor Types (S1)

每个 S1 至少使用以下 3 种干扰类型：

1. **First Suggested Rejected** — 先提选项A但被拒绝，改为B
2. **Number Correction** — 先提一个数字（价格/日期/时间），后更正
3. **Spelling Clarification** — 拼写确认，题目问名字/地址
4. **Date/Price Correction** — 日期或价格被修改
5. **Changed Preference** — "I was thinking X, but actually Y"

### 2.5 语言规范 | Language Norms (S1)

- **禁止**学术词汇（analysis, methodology, hypothesis 等）
- 词汇难度：CEFR A2-B1
- 话轮自然交替，每轮对话 2-4 个话轮
- 包含口语特征：hesitation ("um", "well", "let me think"), fillers ("you know", "actually", "I mean")
- 句子长度平均 8-15 个单词
- 语速适当（TTS 可设置为 normal speed）

### 2.6 示例对比 | Example: Good vs Bad S1

```
❌ 差 (听起来像念稿):
A: Welcome to our language school. Our courses include beginner,
   intermediate, and advanced levels. What level are you interested in?
B: I am interested in the intermediate course which costs $500.

✅ 好 (自然对话):
A: Good morning, City Language School. How can I help?
B: Oh, hi! I'm looking for a language course. I'm not sure which level...
A: Well, we have several levels. Have you studied before?
B: A little, yes. I took classes in high school. So maybe... intermediate?
A: That sounds about right. The intermediate course runs twice a week.
B: Great. And how much is it?
A: The standard fee is $500... Actually, wait — we have an early bird offer this month. It's $400.
B: Oh, that's good. I'll go with that.
→ 答案确定：$400 (非$500)
```

## 3. Section 2 详细规则

### 3.1 场景清单 | Scenario List (S2)

Section 2 是单人说明/信息型独白，场景限于以下：

| 场景 | 说明内容 | 常见题型 |
|------|---------|---------|
| Activity introduction | 活动介绍（节日、比赛、展览） | matching, completion |
| Venue/facility intro | 场地/设施介绍（博物馆、公园、体育馆） | matching, multiple choice |
| Service/organization explanation | 服务/组织说明（志愿者项目、旅游公司） | completion, matching |
| Event schedule | 活动时间安排（会议议程、节日日程） | completion, matching |
| Orientation talk | 迎新/入职说明（学校、公司） | multiple choice, completion |
| Community information | 社区信息（当地服务、设施开放） | multiple choice, matching |

### 3.2 信息组织 | Information Organization

S2 必须采用总-分结构，有明显的分段信号。标准结构：

```
Introduction (1-2 sentences)
├── "Good morning everyone. I'm here to tell you about..."
├── "Today I'll be talking about..."

Main Body (分段递进)
├── "First, let me explain..."  [Part 1]
├── "Another aspect is..."       [Part 2]
├── "Moving on to..."            [Part 3]
└── "Finally, I'd like to..."    [Part 4]

Conclusion (brief)
├── "So to sum up..."
└── "Any questions?"
```

### 3.3 题型规则 | Question Type Rules (S2)

**支持的题型：**

| 题型 | 子类型 | 规则 |
|------|--------|------|
| Matching | 特征匹配 | 选项数 ≥ 待选项数 (extra options serve as distractors) |
| Multiple Choice | A-C | 仅允许 A, B, C 三个选项 |
| Note Completion | 笔记填空 | 每个空 ONEWORD ONLY |
| Table Completion | 表格填空 | 每空 ONEWORD ONLY |
| Sentence Completion | 句子填空 | 最多 TWO WORDS |

**Map tasks 严格禁止。**

### 3.4 信息密度规则 | Information Density Rules (S2)

- 信息密度：高于 S1，低于 S3
- 每 30 秒音频约包含 2-3 个题目的信息点
- 相邻题目的信息点之间至少间隔 5-8 秒
- 无明显 Question-Answer 格式（和 S1 区分）

### 3.5 选项不能照抄原文 | Paraphrase Rule (S2)

对于选择题和配对题：

```
Transcript: "The main hall can hold up to 200 people."
→ 正确选项: "capacity of 200"
→ ❌ 错误选项: "hold up to 200" (照抄原文)
→ ❌ 干扰选项: "holds exactly 200" (改变了量词)
```

每个选项必须至少替换原文中的一个关键表达方式。

### 3.6 示例对比 | Example: Good vs Bad S2

```
❌ 差 (像在读稿):
The first activity is the treasure hunt which takes place in the park.
It costs $10 per person. The second activity is face painting which is
free and happens in the hall.

✅ 好 (自然的说明介绍):
Let's start with the activities. In the morning, we have something for the kids —
a treasure hunt in the park. It's always very popular, so we recommend arriving early.
There's a small fee — just $10 per child. And in the afternoon, we'll move things
to the main hall where there'll be face painting. That one's completely free.
→ 信息自然嵌入叙述，非列表式
```

## 4. Section 3 详细规则

### 4.1 场景清单 | Scenario List (S3)

Section 3 是学术讨论对话（学生之间或师生之间），场景限于：

| 场景 | 讨论内容 | 必须考察的能力 |
|------|---------|---------------|
| 项目讨论 | 研究项目规划、方法、进展 | 提出建议、评估方案、分配任务 |
| 作业讨论 | 论文/报告选题、结构、资料 | 比较观点、表达分歧、修改方案 |
| 实验讨论 | 实验设计、结果分析、问题 | 分析问题、提出解决方案、评价结果 |
| 展示准备 | presentation 主题、内容、分工 | 分工协作、提供反馈、修改意见 |
| 选课/专业讨论 | 课程选择、职业规划 | 评估优劣、权衡取舍、表达偏好 |

### 4.2 必须避免的 | What to AVOID (S3)

```
❌ 学生读稿对话:
Tom: Our project is about climate change. The first section discusses causes.
Sarah: Yes, and the second section discusses effects.
→ 听起来像在念稿，不是自然讨论

❌ 纯事实提取:
Tom: The library opens at 9 AM.
Sarah: It closes at 9 PM.
→ 这是 S1 的内容，不是学术讨论

❌ 一人主导:
Tom: We should do X. Then we need Y. After that, we do Z.
Sarah: OK. [始终同意]
→ 缺乏观点碰撞
```

### 4.3 讨论类型 | Discussion Types (S3)

S3 必须包含以下讨论类型中至少 2 种：

| 类型 | 描述 | 语言特征 |
|------|------|---------|
| **比较项目** | 比较两个或多个选项的优劣 | "X is more... than Y", "Compared to A, B has..." |
| **评估建议** | 对某提议进行批评性评估 | "That's a good idea but...", "The problem with that is..." |
| **分配任务** | 谁做什么，如何分工 | "You could handle...", "I'll take care of..." |
| **分析问题** | 识别和讨论问题 | "The issue is...", "What concerns me is..." |
| **提出方案** | 提出新的解决方案 | "What if we...", "Alternatively, we could..." |
| **表达分歧** | 不同意对方观点 | "I see your point, but...", "I'm not so sure about that" |

### 4.4 语言特征 | Language Features (S3)

必须包含自然的停顿犹豫和修正：

```
Hesitation markers:
"Well... I mean, not exactly."
"I'm not sure but... maybe we should..."
"Actually, that reminds me..."

Reformulation:
"Or rather, what I'm trying to say is..."
"What I mean is... not this part, but the methodology."

Agreement with reservation:
"That's true, but have you considered..."
"I agree with the general idea, however..."
"I think you're right about the data, though the analysis part..."

Layered distractors (MCQ):
A. mentioned but wrong context (提到了但语境不对)
B. partially correct but incomplete (部分正确但不完整)
C. correct but for different question (正确但对应不同题目)
D. stated as opinion not fact (以观点而非事实陈述)
```

### 4.5 选择题设计规则 | MCQ Design Rules (S3)

**不能是纯事实提取型选择题：**

```
❌ 错误设计 (纯事实):
What time does the library open?
A) 8 AM  B) 9 AM  C) 10 AM
→ 这种设计像 S1，不是学术讨论

✅ 正确设计 (Evaluation/Opinion):
What do the students agree about the interview method?
A) It's too time-consuming for the project timeline
B) It provides the richest data for their research
C) It would be better to use a survey instead
→ 需要听懂讨论中的评价性语言
```

### 4.6 MCQ 干扰层次 | MCQ Distractor Levels

每个选择题必须有明确的干扰层次策略：

```
Level 1 — Mentioned Distractor (被提及的干扰项)
    选项中的内容在音频中被提及，但并非针对该问题。

Level 2 — Partial Distractor (部分正确干扰项)
    选项部分内容匹配答案，但整体不对。

Level 3 — Opposite Distractor (相反干扰项)
    选项与正确答案相反（如被否定的观点）。

Level 4 — Out-of-Scope Distractor (超出范围的干扰项)
    选项内容正确但不在问题讨论范围内。
```

## 5. Section 4 详细规则

### 5.1 场景清单 | Scenario List (S4)

Section 4 是学术讲座独白，场景限于以下学术领域：

| 学科领域 | 主题示例 |
|---------|---------|
| 自然科学 | 动物行为、植物生态、地质形成、气候变化 |
| 社会科学 | 城市化、人口迁移、教育方法、经济发展 |
| 人文历史 | 文化传统、建筑设计、音乐历史、语言演变 |
| 工程技术 | 材料科学、能源创新、交通系统、可持续设计 |
| 商业管理 | 市场趋势、领导力理论、组织行为、消费者心理 |

### 5.2 Lecture 结构 | Lecture Structure (S4)

```
Phase 1 — Introduction (1-2 分钟)
├── Greeting / topic announcement
│   "Today I'd like to explore the concept of..."
├── Hook / relevance statement
│   "You may have noticed that..."
└── Overview of lecture structure
    "I'll begin with... then move on to... and finally..."

Phase 2 — Background / Context
├── Foundational concepts
└── Historical/theoretical context
    "To understand this, we first need to..."

Phase 3 — Main Topic (核心部分)
├── Subtopic 1 with examples
│   "The first key aspect is..." "...for instance..."
├── Subtopic 2 with examples
│   "Another important factor..." "...take the case of..."
└── Subtopic 3 (如有)
    "Furthermore..."

Phase 4 — Conclusion
├── Summary of key points
└── Closing remark / implication
    "So in conclusion..."
```

### 5.3 信号词分类 | Signal Word Categories (S4)

讲座必须使用明确的信号词来导向答案位置：

| 信号词类型 | 示例 | 用途 |
|-----------|------|------|
| 主题引入 | "Let's turn to...", "Now considering...", "Regarding..." | 新段落/新话题 |
| 序列 | "First...", "Another...", "Additionally..." | 多个观点 |
| 强调 | "Crucially...", "The key point is...", "Importantly..." | 重点标记 |
| 举例 | "For instance...", "Take the case of...", "Such as..." | 例证 |
| 因果 | "Consequently...", "This leads to...", "As a result..." | 因果关系 |
| 对比 | "In contrast...", "Whereas...", "On the other hand..." | 对比/相反 |
| 结论 | "To sum up...", "Overall...", "In summary..." | 总结 |

### 5.4 答案特征 | Answer Characteristics (S4)

S4 答案必须是：

| 可接受的答案类型 | 不可接受的答案类型 |
|-----------------|-------------------|
| 抽象名词 (abstraction, recognition, sustainability) | 专有名词 (人名、地名、公司名) |
| 过程/机制 (formation, migration, adaptation) | 简单年份 (2010, 2020) |
| 原因/结果 (cause, consequence, impact) | 电话号码 |
| 分类术语 (category, species, genre) | 纯数字 (价格、年龄、数量) |
| 属性特征 (flexibility, durability, sensitivity) | 不需要推理的专有名称 |

**核心规则：答案不应是"念出来就能填"的——必须经过 paraphrase 或需要在语境中理解。**

```
❌ 差: The study was conducted in 2010. 答案: 2010
→ 容易被猜到，不需要理解

✅ 好: The researchers chose coastal wetlands for their investigation.
    → 答案所在点无明显暗示，需理解语义后定位

❌ 差: Dr. Smith's theory is called "cognitive mapping". 答案: cognitive mapping
→ 照读专有名词

✅ 好: This mental process involves recognising landmarks and building a
    mental picture of routes. The phenomenon is known as something like
    an internal guide or direction system — what the researchers term
    "cognitive mapping".
    → 先解释概念后给出名称，需要注意力
```

### 5.5 字数规则 | Word Count Rules (S4)

S4 通常为 ONE WORD ONLY。

```
指令          | 典型 Section | 允许字数
ONE WORD ONLY | S4 (常见)    | 1 word
NO MORE THAN TWO WORDS | S2/S3 | 1-2 words
ONE WORD AND/OR A NUMBER | S1 | 1 word or 1 number
```

## 6. 答案字数规则 | Word Count Rules

| 指令 | Section | 规则 |
|------|---------|------|
| ONE WORD ONLY | S4 (常见) | 必须是1个词 |
| NO MORE THAN TWO WORDS | S2, S3 | 1-2个词 |
| ONE WORD AND/OR A NUMBER | S1 | 1个词 或 1个数字 |
| NO MORE THAN THREE WORDS | S1 (罕见) | 1-3个词 |

### 连字符和复合词 | Hyphenated Words

- Cambridge 官方：hyphenated words count as ONE word (e.g., "well-known")
- 如果连字符词在官方可接受范围，标注时注明

### 数字处理 | Number Handling

- 数字可以写成 numeral (150) 或 word (one hundred fifty)
- 对于 S1，两种写法均可接受
- 在问题和答案中保持统一风格
