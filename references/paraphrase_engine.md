# Paraphrase Engine — IELTS Listening Paraphrase Rules (转述机制库)

> 本文件定义听力和题干之间的转述规则。每种转述类型都基于 Cambridge IELTS 19/20 的真实使用模式。
> 每道题目的 paraphrase 必须至少应用一种下面的转换。

## 1. Noun ↔ Verb Phrase

### Transformation | 转换

```
Transcript (verb phrase): "They investigated the effects of..."
→ Stem (noun): "There was an investigation into..."
```

### Pattern | 模式

```
[verb] + [object]  ⟷  [noun form of verb] + of/into + [object]

investigate → investigation (into)
analyse → analysis (of)
compare → comparison (between/of)
propose → proposal (for/to)
recommend → recommendation (for)
```

### Cambridge Observation | 基于真实剑桥的分析

Cambridge 19 S3 中，音频说 "We could collect the data online"，
题目转述为 "Online data collection is recommended"。
名词化将句中动词转化为 noun phrase。

### Examples | 自写例子

```
1. Transcript: "They decided to expand the program."
   Stem: "There was a decision to expand the program."

2. Transcript: "Students need to register before classes start."
   Stem: "Registration must be completed before the course begins."

3. Transcript: "The team chose to interview local residents."
   Stem: "The choice of interviewees was local residents."

4. Transcript: "Scientists developed a new treatment for allergies."
   Stem: "A new treatment for allergies was developed."

5. Transcript: "The government reduced funding for the arts."
   Stem: "A reduction in arts funding was announced."
```

## 2. Simple ↔ Academic Wording

### Transformation | 转换

```
Transcript (simple): "get better"
→ Stem (academic): "improve" / "enhance" / "optimize"
```

### Pattern | 模式

```
Simple word/phrase  ⟷  Academic term

get better → improve / enhance
get worse → deteriorate / decline
a lot → a significant number / a substantial amount
go up → increase / rise / escalate
go down → decrease / decline / drop
find out → discover / determine / ascertain
use → utilize / employ / implement
show → demonstrate / indicate / reveal
come from → originate from / derive from
change → modify / alter / transform
enough → sufficient / adequate
need → require / necessitate
help → facilitate / assist
stop → cease / terminate / discontinue
```

### Cambridge Observation | 基于真实剑桥的分析

Cambridge 20 S2 中，音频说 "the centre is good for all ages"，
题干转述为 "the facility is suitable for people of all generations"。
基本词 "good for" → "suitable for"；"all ages" → "all generations"。

### Examples | 自写例子

```
1. Transcript: "You need a lot of time to finish the project."
   Stem: "The project requires a substantial time commitment."

2. Transcript: "The new method helped students learn faster."
   Stem: "The new method facilitated accelerated learning."

3. Transcript: "The number of visitors went up by 20%."
   Stem: "There was a 20% increase in visitor numbers."

4. Transcript: "They changed the design after the feedback."
   Stem: "The design was modified following user feedback."

5. Transcript: "The experiment showed that the theory was wrong."
   Stem: "The experiment demonstrated the theory to be incorrect."
```

## 3. Cause ↔ Consequence

### Transformation | 转换

```
Transcript (cause): "Eating too much sugar leads to obesity."
→ Stem (consequence): "Obesity can result from a high-sugar diet."
```

### Pattern | 模式

```
[cause] leads to/triggers/results in [consequence]
⟷
[consequence] is caused by/arises from/is the result of [cause]

Cause → Consequence:
leads to → stems from → is the reason for
is due to → is a result of → is caused by
```

### Cambridge Observation | 基于真实剑桥的分析

Cambridge 19 S4 讲座关于 climate adaptation：
音频说 "Rising sea levels force coastal communities to relocate"，
题目转述为 "Relocation of coastal communities is a consequence of rising sea levels"。

### Examples | 自写例子

```
1. Transcript: "The heavy rain caused flooding in the area."
   Stem: "Flooding in the area was caused by heavy rainfall."

2. Transcript: "Lack of exercise results in poor health."
   Stem: "Poor health stems from a lack of physical activity."

3. Transcript: "Pollution leads to respiratory diseases."
   Stem: "Respiratory diseases are linked to pollution."

4. Transcript: "The new policy reduced traffic congestion."
   Stem: "Traffic congestion decreased as a result of the new policy."

5. Transcript: "Stress affects people's sleep quality."
   Stem: "Poor sleep quality is an effect of stress."
```

## 4. Example ↔ Category

### Transformation | 转换

```
Transcript (examples): "dogs, cats, and rabbits"
→ Stem (category): "domestic pets / household animals"
```

### Pattern | 模式

```
[examples / instances]  ⟷  [category / class / type]

dogs, cats, rabbits → household animals / pets
cars, buses, trains → modes of transport
maths, physics, chemistry → science subjects
factories, offices, shops → types of buildings
lions, tigers, bears → large mammals
```

### Cambridge Observation | 基于真实剑桥的分析

Cambridge 20 S2 关于 museum facilities：
音频说 "There's a hands-on area where children can press buttons and lift flaps"，
题目转述为 "The interactive zone offers tactile activities"。

### Examples | 自写例子

```
1. Transcript: "You need to bring a tent and a sleeping bag."
   Stem: "All camping equipment must be provided."

2. Transcript: "The course covers grammar, vocabulary, and pronunciation."
   Stem: "The course addresses fundamental language skills."

3. Transcript: "They offer swimming, running, and cycling."
   Stem: "The centre provides various athletic activities."

4. Transcript: "The report includes graphs, diagrams, and tables."
   Stem: "The report should contain appropriate visual data."

5. Transcript: "The museum has paintings, sculptures, and photographs."
   Stem: "The museum's collection spans multiple art forms."
```

## 5. Opinion ↔ Evaluative Wording

### Transformation | 转换

```
Transcript: "She thinks the method is too complicated."
→ Stem: "She considers the method to be unnecessarily complex."
```

### Pattern | 模式

```
[person] thinks/believes [opinion]
⟷
[person] considers/regards/views [topic] as [evaluation]

think → consider (to be) / regard (as) / view (as)
like → appreciate / value / favour
dislike → object to / be critical of / question
agree → support / endorse / concur
disagree → challenge / dispute / question
```

### Cambridge Observation | 基于真实剑桥的分析

Cambridge 19 S3 MCQ：
音频说 "Sarah thinks the interview method gives detailed responses"，
题干转述为 "Sarah considers interviewing to provide in-depth data"。

### Examples | 自写例子

```
1. Transcript: "Tom thinks the budget is too small."
   Stem: "Tom considers the budget to be insufficient."

2. Transcript: "The professor likes the research proposal."
   Stem: "The professor views the research proposal favourably."

3. Transcript: "Both students agree with the plan."
   Stem: "The plan was endorsed by both students."

4. Transcript: "Sarah dislikes the questionnaire format."
   Stem: "Sarah is critical of the questionnaire design."

5. Transcript: "I think we should change the topic."
   Stem: "The speaker is in favour of a topic change."
```

## 6. Purpose ↔ Intention

### Transformation | 转换

```
Transcript: "We created this to raise awareness..."
→ Stem: "The purpose of creating this was to increase awareness..."
```

### Pattern | 模式

```
[action] to [purpose]
⟷
The purpose/goal/aim of [action] is [purpose]

to → in order to → with the aim of
for the purpose of → designed to → intended to
the reason for → the rationale behind → the objective is
```

### Cambridge Observation | 基于真实剑桥的分析

音频说 "We started the program to help local businesses grow"，
题干转述为 "The program was initiated with the aim of supporting local business growth"。

### Examples | 自写例子

```
1. Transcript: "The survey was conducted to understand customer needs."
   Stem: "The purpose of the survey was to gauge customer requirements."

2. Transcript: "They built the centre to provide recreational facilities."
   Stem: "The centre was constructed for recreational purposes."

3. Transcript: "The guide was written to help new students."
   Stem: "The guide is intended to assist incoming students."

4. Transcript: "We changed the schedule to avoid conflicts."
   Stem: "The schedule adjustment aimed to prevent timetable clashes."

5. Transcript: "The experiment is designed to test the hypothesis."
   Stem: "The objective of the experiment is hypothesis testing."
```

## 7. Problem ↔ Result

### Transformation | 转换

```
Transcript (problem): "The room was too small for the group."
→ Stem (result): "Lack of space was an issue for the workshop."
```

### Pattern | 模式

```
[problem / issue]  ⟷  [result / consequence / complaint]

too expensive → budget constraints / financial limitations
too small → insufficient space / size limitations
too slow → delays / time constraints / inefficiency
too difficult → challenges / complications / difficulties
not enough → shortage / lack of / insufficient
```

### Cambridge Observation | 基于真实剑桥的分析

Cambridge 20 S3 MCQ：
音频说 "We didn't have enough time to finish the analysis properly"，
题干转述为 "Time constraints affected the quality of the analysis"。

### Examples | 自写例子

```
1. Transcript: "The lab didn't have enough equipment for everyone."
   Stem: "Equipment shortages affected the practical sessions."

2. Transcript: "The venue was too far from the city centre."
   Stem: "The venue location presented accessibility challenges."

3. Transcript: "The software was too complicated for beginners."
   Stem: "Software complexity was a barrier for novice users."

4. Transcript: "The data collection took much longer than expected."
   Stem: "Unexpected delays in data collection impacted the timeline."

5. Transcript: "Not enough people volunteered for the event."
   Stem: "A shortage of volunteers caused difficulties."
```

## 8. Condition ↔ Restriction

### Transformation | 转换

```
Transcript (condition): "If it rains, we'll move the event indoors."
→ Stem (restriction): "The event will be held indoors except in wet weather."
```

### Pattern | 模式

```
If [condition] → [result]
⟷
[result] except in [condition] / unless [condition]

if → on the condition that → provided that
unless → except when → except in
only if → subject to → contingent on
```

### Cambridge Observation | 基于真实剑桥的分析

音频说 "You can bring your camera only if you have a permit"，
题干转述为 "A permit is required for photography"。

### Examples | 自写例子

```
1. Transcript: "You can enter the lab if you wear safety goggles."
   Stem: "Safety goggles are required for lab access."

2. Transcript: "The discount applies if you book before Friday."
   Stem: "The discount is available for early bookings only."

3. Transcript: "Children can join unless the activity is full."
   Stem: "Children's participation is subject to availability."

4. Transcript: "You can borrow books if you have a membership card."
   Stem: "Membership cards are required for borrowing."

5. Transcript: "We'll proceed with the project if funding is approved."
   Stem: "The project will go ahead, subject to funding approval."
```

## 9. Informal ↔ Formal Stem

### Transformation | 转换

```
Transcript (informal/spoken): "There were loads of people."
→ Stem (formal): "A significant number of attendees were present."
```

### Pattern | 模式

```
Informal (spoken)  ⟷  Formal (written stem)

loads of / lots of → a significant number of / a considerable amount
kids → children / young people
fix → repair / resolve
get → obtain / receive / acquire
a bit → somewhat / slightly / to some extent
make up → constitute / comprise
look into → investigate / examine
come up with → propose / devise / formulate
put off → postpone / delay
carry out → conduct / undertake / perform
```

### Cambridge Observation | 基于真实剑桥的分析

Cambridge 19 S1 对话中，Speaker 说 "I'm really into photography"，
题干转述为 "The applicant has a strong interest in photography"。

### Examples | 自写例子

```
1. Transcript: "They spent a load of money on advertising."
   Stem: "A significant budget was allocated to advertising."

2. Transcript: "The team came up with a great idea."
   Stem: "The team proposed an innovative concept."

3. Transcript: "They put off the event until next month."
   Stem: "The event was postponed to the following month."

4. Transcript: "We looked into different options for the project."
   Stem: "Various project options were investigated."

5. Transcript: "The kids really liked the hands-on activities."
   Stem: "The hands-on activities were particularly popular among young participants."
```

## 10. Specific ↔ General

### Transformation | 转换

```
Transcript (specific): "on Tuesdays"
→ Stem (general): "weekly"
```

### Pattern | 模式

```
Specific detail  ⟷  General description

Mondays / Tuesdays → weekdays
7-9 pm → evening
March / April → spring
£10 / £15 → a fee / a charge
30 minutes / 1 hour → a short period / a brief session
walking / cycling → active transport
coffee / tea → hot beverages / refreshments
```

## 11. Positive ↔ Negative Inversion

### Transformation | 转换

```
Transcript (positive): "The museum is only open in the morning."
→ Stem (negative): "The museum is closed in the afternoon."
```

### Pattern | 模式

```
Positive statement  ⟷  Negative equivalent

only open from...to → closed except
available for → not available to / restricted
free → not free / charges apply
included → not included / additional
allowed → prohibited / not permitted
always → never / not always
most → not all
```

### Cambridge Observation | 基于真实剑桥的分析

音频说 "The café is open during weekdays"，
题干转述为 "The café is closed at weekends"。
从正说转为反说。

### Examples | 自写例子

```
1. Transcript: "Only registered students can use the lab."
   Stem: "The lab is not accessible to unregistered individuals."

2. Transcript: "The event is free for children under 12."
   Stem: "Children aged 12 and above must pay for the event."

3. Transcript: "Parking is available for members only."
   Stem: "Non-members do not have access to parking."

4. Transcript: "The exhibition runs throughout the summer."
   Stem: "The exhibition does not run outside the summer months."

5. Transcript: "Guests are welcome at all events."
   Stem: "No events restrict attendance to members exclusively."
```

## 12. Active ↔ Passive

### Transformation | 转换

```
Transcript (active): "The government built the facility in 2010."
→ Stem (passive): "The facility was constructed in 2010."
```

### Pattern | 模式

```
[Subject] [verb] [object]
⟷
[Object] [be verb] [past participle] [by subject/implied]

The team developed the software → The software was developed
Researchers collected data → Data was collected
The centre offers courses → Courses are offered
The report includes data → Data is included in the report
```

### Cambridge Observation | 基于真实剑桥的分析

Cambridge 20 S4：
音频说 "Early farmers developed irrigation systems"，
题干转述为 "Irrigation systems were developed by early farming communities"。

### Examples | 自写例子

```
1. Transcript: "The company funded the research project."
   Stem: "The research project was funded by the company."

2. Transcript: "Scientists discovered the species in 2015."
   Stem: "The species was discovered in 2015."

3. Transcript: "The museum hosts workshops every Saturday."
   Stem: "Workshops are held at the museum each Saturday."

4. Transcript: "The team analysed the survey results."
   Stem: "The survey results were analysed."

5. Transcript: "Local volunteers manage the community garden."
   Stem: "The community garden is managed by local volunteers."
```

## 13. Temporal Shift

### Transformation | 转换

```
Transcript (temporal): "before the exam"
→ Stem (temporal): "prior to the examination"
```

### Pattern | 模式

```
Informal time  ⟷  Formal time

before → prior to → preceding
after → following → subsequent to
during → throughout → in the course of
until → up to → no later than
at the same time → simultaneously → concurrent with
then → subsequently → thereafter
first → initially → at the outset
later → subsequently → at a later stage
```

### Cambridge Observation | 基于真实剑桥的分析

音频说 "Before the course starts, you'll need to read the handbook"，
题干转述为 "Reading the handbook is required prior to course commencement"。

### Examples | 自写例子

```
1. Transcript: "After the lecture, there will be a discussion."
   Stem: "A discussion session will follow the lecture."

2. Transcript: "Students must register during the first week."
   Stem: "Registration must be completed within the initial week."

3. Transcript: "The museum closes at 5 PM."
   Stem: "The museum closes no later than 5 PM."

4. Transcript: "First, we'll review the literature."
   Stem: "The initial step involves a literature review."

5. Transcript: "The results will be published later this year."
   Stem: "Publication of the results is expected at a later date."
```

## 14. Quantifier Change

### Transformation | 转换

```
Transcript: "most students"
→ Stem: "the majority of students / over 50% of students"
```

### Pattern | 模式

```
Informal quantifier  ⟷  Formal quantifier

all → every / the entire / 100% of
most → the majority of / the bulk of
many → a considerable number of / numerous
some → a certain number of / a proportion of
few → a small number of / a minority of
no → none / zero / not any
half → 50% / half of
twice → double / two times
```

### Cambridge Observation | 基于真实剑桥的分析

音频说 "A lot of people prefer the morning session"，
题干转述为 "The majority of participants show a preference for morning sessions"。

### Examples | 自写例子

```
1. Transcript: "Almost all students passed the exam."
   Stem: "The vast majority of students achieved a passing grade."

2. Transcript: "Only a few volunteers came."
   Stem: "A small number of volunteers participated."

3. Transcript: "Half the budget went to marketing."
   Stem: "50% of the budget was allocated to marketing."

4. Transcript: "The number of members doubled last year."
   Stem: "Membership increased twofold in the previous year."

5. Transcript: "No visitors complained about the service."
   Stem: "Zero complaints about the service were received."
```

## 转述组合规则 | Paraphrase Combination Rules

### 每题最少应用 | Minimum Per Question

每道题目的题干必须应用至少 **1 种**转述类型。
推荐应用 **2 种**转述类型以获得足够的难度。

### 组合示例 | Combination Examples

```
Example 1 (Noun↔VP + Academic):
Transcript: "The team decided to look into the issue."
Stem (Noun↔VP): "An investigation into the issue was conducted."
Stem (Academic): "An investigation into the matter was undertaken."

Example 2 (Active↔Passive + Temporal):
Transcript: "The council will review the policy next year."
Stem: "The policy will be reviewed in the upcoming year."

Example 3 (Quantifier + General↔Specific):
Transcript: "We had about 50 people at the opening."
Stem: "Approximately fifty individuals attended the launch event."
```

### 禁止的转述 | Prohibited Transformations

- ❌ 只改变一个单词的同义词替换（test → exam）但不改变句式
- ❌ 只改变单词形式（register → registration）但不改变句式
- ❌ 整句照抄 stem 的 3 个以上连续单词
- ❌ 把完整的 audio transcript 句子变成 stem（不是转述，是摘抄）
